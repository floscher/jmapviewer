Maven repository "releases" for JMapViewer

This repository contains artifacts for the commits, which have a git-tag attached on the master branch.

If you want to browse the contents of this repository, visit https://github.com/floscher/jmapviewer/tree/gh-pages/maven/releases

Usage

Point your build system to the following URL of this Maven repository:
https://floscher.github.io/jmapviewer/maven/releases.

The naming scheme of the artifacts is:
org.openstreetmap:jmapviewer:‹release version number› (e.g. org.openstreetmap:jmapviewer:1.0)
