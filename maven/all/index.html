Maven repository "all" for JMapViewer

This repository contains artifacts for all (buildable) commits on the master branch.

If you want to browse the contents of this repository, visit https://github.com/floscher/jmapviewer/tree/gh-pages/maven/all

Usage

Point your build system to this Maven repository using the following URL:
https://floscher.github.io/jmapviewer/maven/all.

The naming scheme of the artifacts is:
org.openstreetmap:jmapviewer:‹SVN revision›.1-‹short git commit hash› (e.g. org.openstreetmap:jmapviewer:9505.1-abcdef1)
